# Alerting Engine (Go)

## Overview
The `alerts/` folder contains background algorithms to enforce sanity and reduce notification fatigue generated by the AI pipelines. 

## Responsibilities
- **Fatigue Control**: Deduplicates repeated alert outputs resolving to identical temporal features.
- **Escalation Pathways**: Distributes validated alerts across endpoints, webhooks, or log CLI destinations securely.
- **Alert Tuning**: Temporarily dampens alerts dynamically based on tracked baseline drift updates.
